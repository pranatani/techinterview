Power BI Interview Questions

Beginner (33 Questions)

What is Power BI?

Power BI is a Microsoft business analytics tool for data visualization and sharing insights.

What are the key components of Power BI?

Power BI Desktop, Power BI Service, Power BI Mobile, and Power BI Report Server.

What data sources can Power BI connect to?

Excel, SQL Server, Azure, SharePoint, Google Analytics, and many others.

What is a Power BI dashboard?

A dashboard is a single-page canvas to monitor important metrics through visuals.

What is Power Query?

Power Query is a data connection and transformation tool to extract, transform, and load data.

What is DAX and how is it different from Excel formulas?

DAX is more powerful for relational data and analytics than Excel formulas.

Explain CALCULATE function in DAX.

Used to modify filter context for calculations.

How do you optimize report performance?

Limit visuals, optimize data model, reduce columns, use aggregations.

How do you implement advanced Row-Level Security with dynamic roles?

Using DAX expressions to filter data based on user metadata.

Explain the VertiPaq engine in Power BI.

An in-memory columnar database engine supporting compression and fast queries.

How do you troubleshoot slow reports?

Using Performance Analyzer, query diagnostics, and DAX Studio.

Tableau Interview Questions

Beginner (33 Questions)

What is Tableau?

Tableau is a data visualization tool used to convert raw data into an understandable format with interactive dashboards.

What are the different Tableau products?

Tableau Desktop, Tableau Server, Tableau Online, Tableau Prep, Tableau Public.

What types of data connections does Tableau support?

Live connection and Extract connection to various sources like Excel, SQL, Hadoop, cloud databases.

What is a context filter and how does it improve performance?

Context filter generates a temporary table that other filters act upon improving speed.

Explain LOD (Level of Detail) Expressions.

Allow calculations at granularities different from the view's.

Explain Tableau’s Performance Recorder.

A tool to record and analyze workbook performance bottlenecks.
