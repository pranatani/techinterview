Power BI Interview Questions

What is Power BI?

Power BI is a Microsoft business analytics tool for data visualization and sharing insights.

What are the key components of Power BI?

Power BI Desktop, Power BI Service, Power BI Mobile, and Power BI Report Server.

What data sources can Power BI connect to?

Excel, SQL Server, Azure, SharePoint, Google Analytics, and many others.

What is a Power BI dashboard?

A dashboard is a single-page canvas to monitor important metrics through visuals.

What is Power Query?

Power Query is a data connection and transformation tool to extract, transform, and load data.

What is the use of Power BI Desktop?

Used to create reports and data models before publishing to the Power BI service.

Define 'Dataset' in Power BI.

A collection of data imported or connected that reports and dashboards are based on.

What is DAX in Power BI?

DAX stands for Data Analysis Expressions, a formula language for computations.

What are visualizations in Power BI?

Graphs, charts, maps, tables, and cards used to represent data.

What is the difference between a report and a dashboard?

Report contains multiple pages; dashboards are single pages with pinned visuals from reports.

What are slicers?

Slicers are visual filters that allow users to slice data by categories.

What is the Power BI service?

The online SaaS platform where reports and dashboards are published and shared.

Can Power BI connect live to data sources?

Yes, via DirectQuery mode.

What is Power BI Gateway?

A bridge to connect on-premises data sources securely with Power BI Service.

Explain the difference between Import and DirectQuery.

Import loads data into Power BI; DirectQuery queries data live without storing it.

What is a calculated column?

A new column added to a data table using DAX formulas.

What is a measure?

A dynamic calculation evaluated on the fly in reports.

Can you schedule data refresh in Power BI?

Yes, datasets can refresh automatically on a schedule.

What is Row-Level Security (RLS)?

A security feature restricting data access by user roles.

What types of relationships exist in Power BI?

One-to-many, many-to-one, and many-to-many.

What is a Bookmark?

Snapshots of a report's current state to create story-like experiences.

What is Power BI Q&A?

A natural language query tool to ask questions and get answers in visuals.

List some common visual types in Power BI.

Bar chart, line chart, pie chart, matrix, card, scatter chart.

What file formats can Power BI Desktop save to?

.pbix (Power BI Desktop file) and .pbit (template file).

What is Power BI mobile?

Mobile apps for accessing dashboards and reports on mobile devices.

What is a Gateway Cluster?

A group of gateways for load balancing and high availability.

What is the maximum data size Power BI can import?

1 GB per dataset in Power BI Pro; more in Premium.

What are themes in Power BI?

Predefined color palettes to style reports.

What is Decomposition Tree visual?

A visual to break down data by dimension hierarchies.

What is an App Workspace?

A collaborative space for creating and distributing Power BI content.

What is Power BI Report Server?

An on-premises report server for hosting Power BI reports.

What is Drill-through?

Feature to navigate from a summary visual to detailed report pages.

What is a Q&A visual in Power BI?

Allows asking questions in natural language to get visual answers.

What is DAX and how is it different from Excel formulas?

DAX is more powerful for relational data and analytics than Excel formulas.

Explain CALCULATE function in DAX.

Used to modify filter context for calculations.

What is a function to create time intelligence calculations?

Functions like SAMEPERIODLASTYEAR(), TOTALYTD().

Difference between calculated column and measure?

Calculated columns add data to tables; measures calculate on report interaction.

What is filter context in Power BI?

Filters applied to visuals impacting calculation results.

Explain data modeling best practices in Power BI.

Use star schema, avoid many-to-many relationships, reduce cardinality.

What is the difference between LEFT JOIN and INNER JOIN in Power Query?

LEFT JOIN keeps all records from left table; INNER JOIN keeps matching records.

What are the storage modes in Power BI datasets?

Import, DirectQuery, Dual.

How do you optimize report performance?

Limit visuals, optimize data model, reduce columns, use aggregations.

What is the difference between Power BI Pro and Premium?

Pro is per-user license; Premium offers dedicated capacity and enhanced performance.

How can you handle many-to-many relationships in Power BI?

Using bridge tables or composite models.

What is a composite model?

Capability to combine DirectQuery and Import tables in a single model.

Explain incremental refresh in Power BI.

Refresh only recent data instead of full dataset to improve performance.

What are custom visuals?

Third-party or user-developed visuals not included by default.

What is Power BI dataflows?

Reusable data preparation entities managed in the Power BI service.

How do you create a hierarchy in Power BI?

By dragging fields to create parent-child data relationships.

What are calculated tables?

Tables created by DAX based on existing data.

What is the usage of USERELATIONSHIP function?

To activate an inactive relationship temporarily.

How do you use the SWITCH function?

To evaluate expressions and return a value based on the first true condition.

How do you implement dynamic titles in Power BI?

Using DAX measures reflecting user selections.

What is the difference between ALL and ALLEXCEPT in DAX?

ALL removes all filters; ALLEXCEPT keeps filters on specified columns.

How is security managed in Power BI?

Via Row-Level Security and App Workspaces permissions.

What are the Power BI API capabilities?

Embedding content, automation, and management via REST APIs.

How do you refresh data in Power BI Service?

Manually, scheduled refresh, or via API.

What is the use of Synoptic Panel visual?

Allows custom map or image overlays for interactive analysis.

How do you handle data privacy settings?

Through data source privacy levels: Public, Organizational, Private.

Explain bookmarks and their use cases.

Used to create report storytelling and save views.

What are aggregations in Power BI?

Pre-aggregated data to improve query speed.

Explain what is a “What-if” parameter in Power BI.

Allows interactive data scenario testing by what-if analysis.

What is a Power BI template?

A reusable .pbit file with report structure but no data.

How is Power BI integrated with Azure?

Through Azure services like Azure Data Lake, Synapse, and Machine Learning.

What are the limitations of DirectQuery?

Slower performance, restricted DAX, limited transformations.

How do you handle date tables in Power BI?

Create or import a date/calendar table for time intelligence functions.

What is the use of the USERNAME() function?

Retrieve user identity in Row-Level Security rules.

Explain Power BI Premium per capacity licensing.

Dedicated resources shared by multiple users for higher scale.

What is the use of the RELATED function in DAX?

To lookup values from related tables.

How do you manage large datasets in Power BI?

Partitioning, aggregations, incremental refresh, optimizing data models.

How do you implement advanced Row-Level Security with dynamic roles?

Using DAX expressions to filter data based on user metadata.

Explain the VertiPaq engine in Power BI.

An in-memory columnar database engine supporting compression and fast queries.

What is the use of Calculation Groups in Analysis Services?

To simplify and standardize calculations across measures.

How do you use Aggregations with Composite Models?

Pre-aggregated tables redirect queries for speed with direct query fallback.

Explain the Optimizer for DAX Queries.

Rewrites queries and optimizes execution plans.

How can you troubleshoot slow reports?

Using Performance Analyzer, query diagnostics, and DAX Studio.

What are the best practices for data modeling in large scale Power BI?

Star schema, avoiding bi-directional cross filtering, minimizing cardinality.

Explain how to use AI visuals in Power BI.

Using features like Key Influencers, Q&A with AI, and cognitive services.

What is the impact of bi-directional cross-filtering?

Can cause ambiguous relationships and poor performance.

Describe incremental load in Power BI Premium.

Refreshes only a subset of partitions for big datasets.

How do you use the performance analyzer?

To identify bottlenecks in report rendering and query performance.

What is the use of TREATAS in DAX?

Applies the result of a table expression as filters on other tables.

How do you handle calculated tables performance impact?

Avoid excessive recalculations, optimize DAX, use aggregations.

Explain XMLA endpoints in Power BI Premium.

Allows external tools to connect and manage datasets.

What are the limitations of Power BI embedding?

Capacity restrictions, licensing, and API limits.

Explain the differences between Power BI Desktop, Service, and Report Server from a technical perspective.

Desktop is authoring tool; Service is cloud hosting; Report Server is on-premises.

How is data lineage tracked in Power BI?

Using lineage view showing data source impact and dependencies.

How can you connect Power BI with non-Microsoft data sources?

Via connectors, ODBC, REST APIs.

Explain using the 'EXCEPT' function in DAX.

Returns rows from one table not present in another.

What are the various storage optimization techniques?

Column reduction, data type optimization, removing unnecessary tables.

How to create custom connectors for Power BI?

Using Power Query M language and SDK.

Explain the use of Power BI's Composite Models with DirectQuery for multiple sources.

Combining Import tables and DirectQuery sources in one dataset.

What is the role of Azure Analysis Services with Power BI?

Hosting models with higher scalability and enterprise security.

Describe the architecture of Power BI.

Data is ingested via Power BI Desktop, processed by VertiPaq engine, rendered in visuals.

What is DAX Studio used for?

DAX query authoring and performance tuning.

Explain Power BI's data refresh hierarchy.

Dataset refresh, gateway, data source.

What are Personal Gateway and Enterprise Gateway?

Gateway modes for individual users and organizational deployment.

Outline the steps for migrating reports from Power BI Desktop to Service.

Publish directly or via workspace, configure data sources and permissions.

How do you handle real-time streaming in Power BI?

Using push datasets and streaming dataflows.

Discuss best practices for deploying Power BI reports in organizations.

Version control, data governance, security, and training.
